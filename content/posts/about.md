#  JavaScript数组方法合集

1、创建一个数组

```js
// 字面量方式:
// 最常用，在初始化数组的时候 相当方便
var a = [1, 2, 3]; // [1, 2, 3];
// 构造器: Array()
//没有参数时生成空数组
var a = Array(); // [] 
//只添加一个参数时，默认为参数项个，每一项为undefined
var a = Array(3); // [undefined,undefined,undefined]
var a = Array(1,2,3); // [1, 2, 3]
```
